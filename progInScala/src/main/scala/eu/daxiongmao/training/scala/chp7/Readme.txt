This chapter demonstrates some basic controls and loop structures.

Key points:
- Beware the assignation and immediate comparison. You'll compare the resulting TYPES and not the VALUES
- Always prefers VAL instead of VAR (~ java: use final variable only)
- Always prefers FOR instead of WHILE == replace your while(..){} by a recursive for(..){} + val
- You can use match (~ java switch) to assign value

Enjoy!
